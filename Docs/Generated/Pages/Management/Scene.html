class Scene
Manages the creation, persistence, and deletion of SceneObjects in the Hengine. It also manages the physics engine. This class should not be constructed, and can be accessed via the .scene property of both the global object and Hengine. scene.gravity = new Vector2(0, 0.1); // low gravity const floor = scene.main.addPhysicsRectElement("floor", width / 2, height, width, 50); const BLOCK_SIZE = 30; for (let i = 0; i < 10; i++) // generate 10 slightly disorganized blocks that will fall scene.main.addPhysicsRectElement( "block", width / 2 + Random.range(-10, 10), floor.getBoundingBox().y - (i + 0.5) * BLOCK_SIZE, BLOCK_SIZE, BLOCK_SIZE, true );
Properties
mainElementContainer
The root of the element tree for the scene
cameraCamera
The camera used to render the scene
mouseEventsBoolean
Whether or not mouse events will ever be checked. If this is true, specific SceneObjects can opt out, but not vice-versa
cullGraphicsBoolean
Whether or not SceneObject graphics will ever be culled. If this is true, specific SceneObjects can still opt out, but not vice-versa
collisionEventsBoolean
Whether or not collision events will be detected
updatingBoolean
Whether the scene is in the process of updating the SceneObjects
renderOrderSceneObject[]
A list of all of the objects most recently rendered, in the order they were rendered in. This updates prior to rendering each frame. This property is read-only.
Methods

Sets the gravitational acceleration for the physics engine.
Parameters

The new gravitational acceleration
get gravityVector2
Returns the current gravitational acceleration for the physics engine. This is initially new Vector2(0, 0.4).
get constraintsConstraint[]
Returns all the active constraints in the scene.
rayCast(rayOrigin, rayDirection, mask?){ hitShape: SceneObject, hitPoint: Vector2 }/null
Performs a ray-cast against some or all of the SceneObjects in the scene. If the ray-cast fails, then null is returned. Otherwise returns an object with a .hitShape property specifying which SceneObject was hit, and a .hitPoint property containing the point of intersection between the ray and the SceneObject.
Parameters
rayOrigin Vector2
The origin point of the ray
rayDirection Vector2
The direction of the ray
mask (SceneObject) => Boolean
A filter for which SceneObjects should be considered in the ray-cast. Default is (object) => true
