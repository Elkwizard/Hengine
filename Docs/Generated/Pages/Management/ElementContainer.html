class ElementContainer extends SceneElement
Represents a non-leaf node in the element tree, and can contain SceneObjects or additional ElementContainers. All methods on this class that add SceneObjects only take effect at the end of the update cycle.
Properties
defaultScriptClass
The default element script applied to all SceneObjects in the container upon creation
