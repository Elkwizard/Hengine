class InputHandler
Provides an API for interacting with a key-based input device (mouse, keyboard). This is an abstract superclass and should not be constructed.
Subclasses
KeyboardHandler, MouseHandler, TouchHandler

class InputHandler.State
Represents the current state of a key on an input device. This is an abstract superclass and should not be constructed. All properties of this class are read-only.
Subclasses
MouseHandler.State
Properties
handlerInputHandler
The handler associated with the input device.
nameString
The name of the associated key.
Methods
get pressedBoolean
Returns whether the key is currently pressed.
get releasedBoolean
Returns whether the key is not currently pressed.
get justPressedBoolean
Returns whether the key became pressed within the last frame.
get justReleasedBoolean
Returns whether the key stopped being pressed within the last frame.
get pressLengthNumber
Returns the duration of the current press of the key. If the key is not currently pressed, this returns 0.
get releaseLengthNumber
Returns the amount of frames since the key was released. If the key is currently pressed, this returns 0.
