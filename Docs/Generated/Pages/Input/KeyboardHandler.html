class KeyboardHandler extends InputHandler
Represents the API for interacting with the user's keyboard. The names of letter keys should always be lowercase, and the names of special keys are same as the identifiers used for KeyboardEvent.key. This class is available via the .keyboard property of both the global object and HengineLoader. intervals.continuous(() => { // change circle color based on whether the space bar is pressed const color = keyboard.pressed(" ") ? new Color("red") : new Color("blue"); renderer.draw(color).circle(width / 2, height / 2, 50); });
