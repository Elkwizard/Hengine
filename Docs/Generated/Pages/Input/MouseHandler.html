class MouseHandler extends InputHandler
Represents the API for interacting with the user's mouse. The names of keys for the mouse are Left, Middle, and Right, for the associated buttons. This class is available via the .mouse property of both the global object and Hengine. All properties of this class are read-only. intervals.continuous(() => { // display a circle at the cursor position when pressing the left mouse button if (mouse.pressed("Left")) renderer.draw(new Color("red")).circle(mouse.screen, 10). });
Properties
wheelDeltaNumber
The scroll displacement (in pixels) from the mouse wheel during the last frame.
screenVector2
The current cursor position, in screen-space
screenLastVector2
The cursor position last frame, in screen-space
worldVector2
The current cursor position, in world-space
worldLastVector2
The cursor position last frame, in world-space
Methods
screenDragState/screenDragEnd/worldDragStart/worldDragEnd(name)Vector2
Returns a location associated with a given key.
Parameters
name String
The name of the key to check
class MouseHandler.State extends InputHandler.State
The state of a given key on a mouse. All properties of this class are read-only.
Subclasses
TouchHandler.State
Properties
screenDragStartVector2
The beginning of the most recent click-and-drag motion with this key, in screen-space
screenDragEndVector2
The end of the most recent click-and-drag motion with this key, in screen-space. If such a gesture is ongoing, this will be the current cursor position
worldDragStartVector2
The beginning of the most recent click-and-drag motion with this key, in world-space
worldDragEndVector2
The end of the most recent click-and-drag motion with this key, in world-space. If such a gesture is ongoing, this will be the current cursor position
