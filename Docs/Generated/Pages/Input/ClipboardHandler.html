class ClipboardHandler
Represents the API for interacting with the user's clipboard. This class should not be constructed directly, and can be accessed from the .clipboard property of both the global object and Hengine. intervals.continuous(() => { if (mouse.justPressed("Middle")) { // copies a message when middle clicking clipboard.write("Hello World!"); } });
Methods
write(text | image)void
Writes a value to the clipboard.
Parameters
text String
The text to write to the clipboard
Parameters
image ImageType
The image to write to the clipboard
read()String
Returns the text content of the clipboard. This value only updates after paste operations.
