class TEXT_AREA extends ElementScript
Makes a SceneObject into a text editor. This only works on 2D SceneObjects, which includes only WorldObjects in 2D Mode. Since the entire screen is a <canvas> in the Hengine, this serves as an alternative to <textarea>. // single line text-box const textbox = scene.main.addUIElement("textbox", width / 2, height / 2, 500, 50); textbox.scripts.add(TEXT_AREA, Font.Arial40, false); textbox.scripts.TEXT_AREA.alwaysIgnore("Enter"); intervals.continuous(() => { if (keyboard.justPressed("Enter")) console.log(textbox.scripts.TEXT_AREA.value); });
Properties
value String
The current content of the text area
immutable

The initial index of the selected text
selectionEnd Number
The first index not in the selected text. If nothing is selected, this will be equal to .selectionStart

Whether the text area should allow new lines and scrolling

Whether this is the currently focused text area

The font used to display the text area
caretColor Color

highlightColor Color
The semi-transparent background color of selected text
padding Number
The interior padding of the text area, measured as a proportion of the font size
scrollBarSize Number
The width of the scroll bars of a multiline text area
scrollSpeed Number
The speed (in pixels per frame) of the scroll bars of a multiline text area
scrollOffset Vector2
The translation of the visible area of the text area due to scrolling

Whether the text area can have multiple lines and scrolling. Default is true
