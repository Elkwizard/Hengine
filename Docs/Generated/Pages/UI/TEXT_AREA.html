class TEXT_AREA extends ElementScript
Makes a SceneObject into a text editor. Since the entire screen is a <canvas> in the Hengine, this serves as an alternative to <textarea>. // single line text-box const textbox = scene.main.addUIElement("textbox", width / 2, height / 2, 500, 50); textbox.scripts.add(TEXT_AREA, Font.Arial40, false); textbox.scripts.TEXT_AREA.alwaysIgnore("Enter"); intervals.continuous(() => { if (keyboard.justPressed("Enter")) console.log(textbox.scripts.TEXT_AREA.value); });
Properties
valueString
The current content of the text area. This value is read-only
selectionStartNumber
The initial index of the selected text
selectionEndNumber
The first index not in the selected text. If nothing is selected, this will be equal to .selectionStart
multilineBoolean
Whether the text area should allow new lines and scrolling
focusedBoolean
Whether this is the currently focused text area
fontFont
The font used to display the text area
caretColorColor
The color of the caret. Starts as black
highlightColorColor
The semi-transparent background color of selected text
paddingNumber
The interior padding of the text area, measured as a proportion of the font size
scrollBarSizeNumber
The width of the scroll bars of a multiline text area
scrollSpeedNumber
The speed (in pixels per frame) of the scroll bars of a multiline text area
scrollOffsetVector2
The translation of the visible area of the text area due to scrolling
Methods
init(font, paddingEM?, multiline?, renderText?)void
Adds a text editor to a SceneObject.
Parameters
font Font
The font used for rendering the text
paddingEM Number
The interior padding of the text area, measured as a proportion of the font size. Default is 0.5
multiline Boolean
Whether the text area can have multiple lines and scrolling. Default is true
renderText (String, Font, Vector2, (Number) => Vector2, Number) => void
A function used to render the text. The default renders the text in place with a black color
alwaysIgnore(key)void
Forces the text area to ignore a specific key input.
Parameters
key String
The name of the key to ignore
setValue(value)void
Sets the content of the text area.
Parameters
value String
The new content for the text area
focus()void
Forcibly focuses the text area.

Forcibly un-focuses the text area, without focusing a different one.
