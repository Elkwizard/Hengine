class DRAGGABLE extends ElementScript
Makes a SceneObject draggable with the mouse. If the object has PHYSICS, then it will become stationary while being dragged. This forces the object to accept mouse events. // create a draggable object with no bounds const object = scene.main.addUIElement("box", width / 2, height / 2, 200, 100); object.scripts.add(DRAGGABLE, "Left");
Properties
keyString
The mouse button that can be used for dragging
draggedBoolean
Whether the object is currently being dragged
boundsRect/null
The bounds in which the object can be dragged
Methods
init(key, bounds?)void
Makes the object draggable with the mouse.
Parameters
key String
The mouse button that can be used to drag the object
bounds Rect
The bounds in which the object can be dragged. Default is null
