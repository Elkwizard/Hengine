class Animatable
Represents a MathObject-valued variable that smoothly moves between values. If a transition between states is interrupted by setting a new target, the new transition will begin immediately and will begin from the current position. const point = new Animatable(middle, 100, Interpolation.smooth); intervals.continuous(() => { if (mouse.justPressed("Left")) point.target = mouse.screen; renderer.draw(new Color("black")).circle(point.value, 10); });
Properties
targetOperable
The current target value of the animatable
durationNumber
The length of each transition, in frames
easing(Number) => Number
The easing function for the transitions
copyTargetBoolean
Whether or not target values should be copied. If this value is false, changing the value passed into target will change the trajectory of the value, even if the value is not passed in again
