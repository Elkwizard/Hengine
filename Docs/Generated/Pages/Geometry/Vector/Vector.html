class Vector extends Operable
Represents a multidimensional vector. This class is abstract and should not be constructed.
Subclasses
Vector2, Vector3, Vector4
Properties
magNumber
The magnitude of the vector
sqrMagNumber
The squared magnitude of the vector
normalizedVector
The unit vector in the same direction of the vector
inverseVector
The vector in the opposite direction with the same magnitude
Methods
invert()Vector
Inverts the vector in-place and returns the caller.
normalize()Vector
Normalizes the vector in-place and returns the caller.
dot(other)Number
Computes the dot product between the caller and another vector.
Parameters
other Vector
The vector to take the dot product with
projectOnto(other)Vector
Computes the projection of the caller onto another vector. Doesn't mutate the caller.
Parameters
other Vector
The vector to project the caller onto
toString()String
Converts the vector to a human readable String representation.
Vector.dist(a, b)Number
Computes the distance between two vectors.
Parameters
a Vector
The first vector
b Vector
The second vector
Vector.sqrDist(a, b)Number
Computes the squared distance between two vectors.
Parameters
a Vector
The first vector
b Vector
The second vector
