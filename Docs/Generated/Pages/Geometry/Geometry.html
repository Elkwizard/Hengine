class Geometry
Provides a collection of 2D geometric algorithms that operate on shapes and vectors. All methods of this class are static and do not mutate their arguments.

Geometry.rayCast(rayOrigin, rayDirection, shapes)Object/null
Returns the closest intersection of a ray with a collection of shapes. The return value is either null (if the raycast misses) or an object with two properties: a .hitPoint property containing the location of the ray intersection, and a .hitShape property containing the shape that the ray intersected.
