class Shape implements Copyable
Represents a solid shape. Within the documentation of this class, Vector refers to either Vector2 or Vector3 depending on whether the 2D or 3D subclass is used. Similarly, Matrix refers to either Matrix3 or Matrix4. This is an abstract superclass and should not be constructed. All properties of this class are read-only.
Subclasses
Shape2D, Shape3D
Properties
