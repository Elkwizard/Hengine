class Polyhedron extends Shape3D
Represents a closed 3D polyhedron as a collection of contiguous triangles.
Subclasses
Prism, Frustum
Properties
verticesVector3[]
The vertices of the polyhedron
indicesNumber[]
An array representing the triangles of the polyhedron. Every three indices in the array specify the locations in the .vertices array which make up a triangle
Methods
getEdges()Line3D[]
Returns a set of unique lines that represent the edges of the faces of the polyhedron.
getFaces()Triangle[]
Returns a set of triangles making up the surface of the polyhedron.
subdivide(count?)Polyhedron
Subdivides each triangle in the polyhedron into a power of 4 number of additional triangles. The subdivided mesh is a copy and is returned.
Parameters
count Number
The power of 4 to multiply the triangle count by. Default is 1
map(vertexTransform)Polyhedron
Creates a new polyhedron by transforming every vertex using a given function.
Parameters
vertexTransform (Vector3) => Vector3
The function to apply to each vertex
Polyhedron.fromTriangles(triangles)Polyhedron
Creates a new polyhedron from a list of triangles which form a closed surface.
Parameters
triangles Triangle[]
The triangles to use as the surface
Polyhedron.fromSphere(sphere, subdivisions?)Polyhedron
Creates a polyhedron that resembles a sphere to a given level of precision.
Parameters
sphere Sphere
The sphere to approximate
subdivisions Number
The amount of times to apply .subdivide() to the sphere. Default is 2
Polyhedron.fromCylinder(cylinder, steps?)Polyhedron
Creates a polyhedron that resembles a cylinder to a given level of precision.
Parameters
cylinder Cylinder
The cylinder to approximate
steps Number
The amount of vertices to use around the circular faces. Default is 8
