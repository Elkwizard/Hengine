class Polyhedron extends Shape3D
Represents a closed 3D polyhedron as a collection of contiguous triangles.
Subclasses
Prism, Frustum
Properties
verticesVector3[]
The vertices of the polyhedron
indicesNumber[]
An array representing the triangles of the polyhedron. Every three indices in the array specify the locations in the .vertices array which make up a triangle
Methods
constructor(vertices, indices)
Creates a new polyhedron.
Parameters
vertices Vector3[]
The vertices of the polyhedron
indices Number[]
The vertex indices for the triangular faces of the polyhedron
getEdges()Line3D[]
Returns a set of unique lines that represent the edges of the faces of the polyhedron.
getFaces()Triangle[]
Returns a set of triangles making up the surface of the polyhedron.
subdivide(count?)Polyhedron
Subdivides each triangle in the polyhedron into a power of 4 number of additional triangles. The subdivided mesh is a copy and is returned.
Parameters
count Number
The power of 4 to multiply the triangle count by. Default is 1
map(vertexTransform)Polyhedron
Creates a new polyhedron by transforming every vertex using a given function.
Parameters

The function to apply to each vertex
Polyhedron.fromTriangles(triangles)Polyhedron
Creates a new polyhedron from a list of triangles which form a closed surface.
Parameters
triangles Triangle[]
The triangles to use as the surface
Polyhedron.fromSphere(sphere, steps?)Polyhedron
Creates a polyhedron that resembles a sphere to a given level of precision.
Parameters
sphere Sphere
The sphere to approximate
steps Number

Polyhedron.fromCapsule(capsule, steps?)void
Creates a polyhedron that resembles a capsule.
Parameters
capsule Capsule
The capsule to approximate
steps Number
The amount of vertices to use around the circular cross-sections of the capsule. Default is 8
Polyhedron.fromCone(axis, radius, steps?)void
Creates a polyhedron that resembles a right circular cone to a given level of precision
Parameters
axis Line3D
A line segment from the center of the cone's base to the peak
radius Number

steps Number

Polyhedron.fromCylinder(longAxis, radius, steps?)Polyhedron
Creates a polyhedron that resembles a right circular cylinder to a given level of precision.
Parameters
longAxis Line3D
A line segment between the centers of the two circular faces
radius Number
The radius of the cylinder
steps Number
The amount of vertices to use around the circular faces. Default is 8
class PolyhedronBuilder
Represents an append-only structure for building procedural polyhedra. Vertices and triangles can be added through various means, and can at any point be converted into a Polyhedron.
Properties
verticesVector3[]
The vertices of the polyhedron. These can be added to directly, if desired
indicesNumber[]
The indices of the vertices of the triangular faces of the polyhedron. These can be added to directly, if desired
Methods
constructor()
Creates a new building structure, with no vertices or indices.
get polyhedronPolyhedron
Creates a new Polyhedron out of the current state of the builder. The Polyhedron is disconnected, in the sense that future changes to the builder will not be reflected in the created Polyhedron.
addVertex(vertex)Number
Adds a new vertex, and returns its index.
Parameters
vertex Vector3
The vertex to add
addVertices(vertices)Number
Adds a new sequence of vertices, and returns the index of the first vertex in the sequence
Parameters
vertices Vector3[]
The vertices to add
addPolyhedron(polyhedron, flip?)Number
Adds the entire contents of a Polyhedron to the builder, and returns the index of the first vertex added.
Parameters
polyhedron Polyhedron
The Polyhedron to add
flip Boolean
Whether to flip all the faces in the added Polyhedron. This does not modify the first argument in-place. Default is false
addConvexVertices(vertices)Number
Adds a list of vertices representing the edge of a convex polygon, and adds triangles in a "triangle fan" formation to connect them. Returns the index of the first vertex added.
Parameters
vertices Vector3[]
The vertices of the convex polygon
