class Interpolation
Provides a collection of interpolation algorithms that operate on operables and numbers. All methods of this class are static and do not mutate their arguments.
Methods
Interpolation.lerp(a, b, t)Number
Linearly interpolates between two values.
Parameters
a Number
The initial value
b Number
The final value
t Number
The progress proportion from the initial value to the final value, on the interval [0, 1]
