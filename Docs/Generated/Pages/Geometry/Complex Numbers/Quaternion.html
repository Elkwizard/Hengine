class Quaternion extends Complex
Represents a quaternion, a 4D extension of the imaginary numbers. Each quaternion is a number of the form a + bi + cj + dk. These can be used to represent 3D rotations via Quaternion.fromRotation() and Quaternion.prototype.rotate().
Properties
realNumber
The real component of the quaternion
imaginaryVector3
The imaginary components of the quaternion. Each element of this vector corresponds to a coefficient on one of the imaginary units
angleVector3
The quaternion's axis of rotation multiplied by its angle of rotation. Only defined for rotation-representing quaternions
Methods
constructor(real, imaginary?)
Creates a new quaternion.
Parameters
real Number
The real component of the quaternion.
imaginary Vector3
The imaginary components of the quaternion. Default is (0, 0, 1).
rotate(vector, dst?)Vector3
Returns the imaginary components of the product of the caller, the provided vector, and the caller's conjugate (q(0, v)q*). For quaternions created with Quaternion.fromRotation(), this corresponds to a 3D rotation.
Parameters
vector Vector3
The vector to rotate
dst Vector3
The destination to store the resulting vector in. If this is not specified, a new vector will be created
toMatrix(result?)Matrix3
Returns a matrix such that for a Quaternion q and Vector3 v, q.rotate(v) is the same as q.toMatrix().times(v).
Parameters
result Matrix3
The destination to store the resulting matrix in. If this is not specified, a new matrix will be created
get Quaternion.iQuaternion
Returns the first quaternion unit, i.
get Quaternion.jQuaternion
Returns the second quaternion unit, j.
get Quaternion.kQuaternion
Returns the third quaternion unit, k.
Quaternion.fromRotation(axis, angle | rotation)Quaternion
Returns a unit quaternion q such that for a Vector3 v, q.rotate(v) refers to v rotated counter-clockwise about the given axis by the given angle.
Parameters
axis Vector3
The normalized axis of rotation
angle Number
The counter-clockwise angle of rotation
Parameters
rotation Vector3
A vector with length equal to the angle of rotation and direction equal to the axis of rotation
Quaternion.compose(angles)Vector3
Returns the result of applying a series of rotations in sequence. The order is opposite that of matrix multiplication, in the sense that the array [angle1, angle2, angle3] represents rotating first by angle1, then by angle2, and finally by angle3. Returns the final orientation after applying all the provided rotations to an identity orientation.
Parameters
angles Vector3[]
The rotations to apply
