class Transform extends Matrix3
Represents a 2D affine transformation with no scaling. It is composed of a rotation about the origin followed by a translation. const obj = scene.main.addElement("my first object", 0, 0); console.log(obj.transform); // { position: (0, 0), rotation: 0 } obj.transform.rotateAbout(new Vector2(0, 100), Math.PI); console.log(obj.transform); // { position: (0, 200), rotation: Math.PI }
