class Vector2 extends Vector
Represents a 2D vector. const i = new Vector2(1, 0); const j = new Vector2(0, 1); console.log(i.dot(j)); // 0 console.log(i.cross(j)); // 1 console.log(i.plus(j).normalize()); // (1/√2, 1/√2)
Properties
