class Vector extends Operable
Represents a multidimensional vector. This is an abstract superclass and should not be constructed.
Subclasses
Vector2, Vector3, Vector4, Number
Properties
magNumber
The magnitude of the vector
sqrMagNumber
The squared magnitude of the vector
normalizedVector
The unit vector in the same direction of the vector
Methods
normalize()Vector
Normalizes the vector in-place and returns the caller.
dot(other)Number
Computes the dot product between the caller and another vector.
Parameters
other Vector
The vector to take the dot product with
reflect(normal)Vector
Computes the reflection of the caller off a provided normal vector. Doesn't mutate the caller.
Parameters
normal Vector
The normal vector to reflect off of
projectOnto(other)Vector
Computes the projection of the caller onto another vector. Doesn't mutate the caller.
Parameters
other Vector
The vector to project the caller onto
without(axis)Vector
Computes a version of the caller with one coordinate axis replaced with 0. This is equivalent to this.minus(this.projectOnto(axis)) for axis.mag === 1. Doesn't mutate the caller.
Parameters
axis Vector
The unit vector axis to remove from the caller
get Vector.sizeNumber
Returns the size of the vector type.
Vector.dist(a, b)Number
Computes the distance between two vectors.
Parameters
a Vector
The first vector
b Vector
The second vector
Vector.sqrDist(a, b)Number
Computes the squared distance between two vectors.
Parameters
a Vector
The first vector
b Vector
The second vector
