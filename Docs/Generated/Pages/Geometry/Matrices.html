class Matrix extends Float64Array implements Copyable, Serializable
Represents an N by N square matrix which can be used to transform vectors of dimension N or (N - 1) via homogenous coordinates. This is an abstract superclass and should not be constructed.
Subclasses
Matrix2, Matrix3, Matrix4, Camera
Methods
constructor(...elements | ...columns | base?)
Creates a new Matrix. Since this class is abstract, this constructor can only be used via its subclasses.
Parameters
elements Number[]
The elements of the matrix, in row-major order
Parameters
columns Vector[]
The columns of the matrix
Parameters
base Matrix
A matrix of less or equal dimension. If specified, the constructed matrix will be an identity matrix with the elements of this argument superimposed on it from the upper-left. Otherwise, the constructed matrix will be an unaltered identity matrix
get maxScaleFactorNumber
Returns the maximum length ratio (this.times(v).mag / v.mag) that is possible for a Vector v of the same dimension as the caller.
get maxHomogenousScaleFactorNumber
Returns the maximum length ratio (this.times(v).mag / v.mag) that is possible for a Vector v of one less dimension than the caller.
get transposedMatrix
Returns a transposed copy of the caller.

Returns a inverted copy of the caller, or null if the caller is singular.

Returns the determinant of the caller.

class Matrix3 extends Matrix
Represents a 3 by 3 matrix for use with 2D vectors in homogenous coordinates or 3D vectors in standard coordinates. const transformation = Matrix3.mul([ Matrix3.translation(10, 5), Matrix3.rotation(Math.PI) ]); const initialPoint = new Vector2(10, 20); const finalPoint = transformation.times(initialPoint); console.log(finalPoint); // (0, -15) const initialAgain = transformation.inverse.times(finalPoint); console.log(initialAgain); // (10, 20)
