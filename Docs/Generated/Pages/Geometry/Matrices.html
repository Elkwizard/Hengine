class Matrix extends Float64Array implements Copyable, Serializable
Represents an N by N square matrix which can be used to transform vectors of dimension N or (N - 1) via homogenous coordinates. This is an abstract superclass and should not be constructed.
Subclasses
Matrix2, Matrix3, Matrix4
Methods

A matrix of less or equal dimension. If specified, the constructed matrix will be an identity matrix with the elements of this argument superimposed on it from the upper-left. Otherwise, the constructed matrix will be an unaltered identity matrix

class Matrix2 extends Matrix

class Matrix3 extends Matrix
Represents a 3 by 3 matrix for use with 2D vectors in homogenous coordinates or 3D vectors in standard coordinates. const transformation = Matrix3.mulMatrices([ Matrix3.translation(10, 5), Matrix3.rotation(Math.PI) ]); const initialPoint = new Vector2(10, 20); const finalPoint = transformation.times(initialPoint); console.log(finalPoint); // (0, -15) const initialAgain = transformation.inverse.times(finalPoint); console.log(initialAgain); // (10, 20)
