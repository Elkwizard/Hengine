class String
The built-in String class has some additional utility methods in the Hengine.
Methods
capitalize()String
Returns a copy of the caller with the first character capitalized.
cut(boundary)String[2]
Returns the caller split along the first instance of a substring. If the substring is not found, it will return an array containing the caller as the first element and the empty string as the second.
Parameters
boundary String
The substring to split on
indent()String
Returns the caller with each line indented by a single tab character.
