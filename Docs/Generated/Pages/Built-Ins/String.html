class String implements Serializable
The built-in String class has some additional utility methods in the Hengine.
Methods
capitalize()String
Returns a copy of the caller with the first character capitalized.
cut(boundary)String[2]
Returns the caller split along the first instance of a substring. If the substring is not found, it will return an array containing the caller as the first element and the empty string as the second.
Parameters
boundary String
The substring to split on
download(name?, extension?)Promise<void>
Downloads the contents of the string as a .txt file. Returns a promise that resolves when the file downloads.
Parameters
name String
The base name of the file to download. The resulting file with be named by this argument, followed by the extension argument. Default is "string"
extension String
The extension for the downloaded file. Default is "txt"
indent()String
Returns the caller with each line indented by a single tab character.
