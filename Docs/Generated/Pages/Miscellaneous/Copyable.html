interface Copyable
Classes that implement this interface can be copied with perfect fidelity.
Implementers
Matrix, MathObject, Animation, Font, Frame, Range, Shape, Texture, Transform, ByteBuffer
Methods
get(destination?)Copyable
Creates a copy of the object and optionally stores it in a provided destination.
Parameters
destination Copyable
The destination to copy the object into. This must be the same type as the caller
