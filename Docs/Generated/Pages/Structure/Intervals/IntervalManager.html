class IntervalManager
Manages the update loop of the Hengine. This class is available via the .intervals property of the global object. // display FPS data intervals.continuous(() => { renderer.image(intervals.fpsGraph).default(10, 10); });
Properties
rawFpsNumber
The FPS based only on the duration of the last frame. This value is read-only
fpsGraphGraphPlane
A graph of the FPS for the last 400 frames. This value is read-only
frameCountNumber
The total number of frames that have elapsed thusfar. This value is read-only
performanceDataBoolean
Whether or not the interval manager should collect performance data (.fps, .fpsGraph, etc.)
Methods
set fps(fps)
Sets the target/maximum update cycles per second.
Parameters
fps Number
The new target FPS
get fpsNumber
Returns the current number of update cycles per second.
get pausedBoolean
Returns whether or not the update loop is currently paused.
makeGraphPlane(graphs, frameLimit?)GraphPlane
Creates a new GraphPlane.
Parameters
graphs Graph[]
The graphs to display on the plane
frameLimit Number
The number of frames to be graphed at once. Default is 400
pause()void
Stops the update loop.
play()void
Resumes the update loop if this function has been called as many times as .pause().

Parameters
fn Function
The function to be executed every frame

Parameters

frames Number

type Symbol

Animates the value of a Operable or Number property from its current value to another over a specified interval. Returns a promise that resolves when the animation completes.
Parameters
object Object
The object which has the animated property
property String/Symbol
The key of the animated property
finalValue Operable/Number
The value to animate to
duration Number
The duration of the animation
curve Function

type Symbol
When during the update cycle to update the animation. Default is IntervalFunction.BEFORE_UPDATE
