class MovingAverage
Stores a moving average over a sequence of numeric samples. Samples can be added at any time, and current average can be retrieved for no cost.
Properties
average Number
The current average value. This will be 0 if no samples have been provided
immutable
Methods
constructor(samples?)
Creates a new moving average.
Parameters
samples Number
The number of samples (window size) of the average. Default is 10
get lastNumber
Returns the value of the most recent sample. The behavior is undefined if no samples have been added.
clear()void
Clears the content of the averaging window, resetting the object.
add(sample)Number
Adds a new sample value. Returns the new average
Parameters
sample Number
The sample to add to the average
