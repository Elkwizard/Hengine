class HengineAnimationResource extends HengineResource
Represents an external animation to be loaded. When this resource loads, it will resolve to an Animation.
Properties
srcString
The path to a folder containing all the animation frames, named 1.png to n.png
framesNumber
The number of frames in the animation
delayNumber
The number of runtime frames to display each frame for
loopsBoolean
Whether or not the animation loops after completing
Methods
constructor(src, frames, delay, loops)
Creates a new HengineAnimationResource.
Parameters
src String
The path to a folder containing the animation frames
frames Number
The number of frames in the animation
delay Number
The number of runtime frames to display each frame for
loops Boolean
Whether or not the animation loops
