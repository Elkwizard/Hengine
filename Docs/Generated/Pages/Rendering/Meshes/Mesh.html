class Mesh extends Renderable
Represents a 3D mesh composed of chunks with different materials. The following vertex attributes are supported:
Attribute NameMeaningType
vertexPositionThe model-space location of the vertexVector3
vertexUVThe UV texture coordinates of the vertexVector2
vertexNormalThe normalized normal vector of the vertexVector3
Properties
attributes String[]
The vertex attributes present in the mesh's data
immutable
data Float32Array
The vertex data of the mesh, interleaved in the order specified by .attributes
chunks MeshChunk[]
The chunks of the mesh, describing the layout of mesh's faces
vertices ArrayLike
A list of each vertex in the vertex data. Each element of this array has a property for each attribute of the mesh. This property is synchronized with .data
stride Number
The number of elements each vertex takes up in the data array
immutable
offsets Map
A map from attribute names to their offset into each vertex in the data array
immutable
Methods
constructor(attributes, data, chunks)
Creates a new mesh.
Parameters
attributes String[]
The vertex attributes in the mesh data
data Float32Array/ArrayLike
The vertex data of the mesh, either as a float buffer or a list of vertex objects
chunks MeshChunk[]
The chunks of the mesh
get()Mesh
Creates a deep copy of the mesh.
getBoundingBox()Prism
Returns the smallest Prism that contains all the vertexPositions of the mesh.
getBoundingBall()Sphere
Returns a small Sphere that contains all the vertexPositions of the mesh.
transform(transform)Mesh
Transforms the vertexPosition of the mesh in-place by a given matrix and flushes the changes. Returns the caller.
Parameters
transform Matrix4
The homogenous transformation to apply to each vertex position
toPolyhedron()Polyhedron
Creates a Polyhedron that has the same shape as the mesh, using the vertexPosition attribute.
Mesh.fromShape(shape, material?, resolution?)Mesh
Creates a new mesh from a Shape3D, inferring many settings to produce generally acceptable results.
Parameters
shape Shape3D
The shape to create a mesh for
material Material
The material to use for the mesh. Default is Material.DEFAULT
resolution Number
An relevant amount of vertices to use. Default is 8
Mesh.fromPolyhedron(polyhedron, material?, settings?)Mesh
Creates a new mesh from a Polyhedron.
Parameters
polyhedron Polyhedron
The object to use for the mesh
material Material
The material for the mesh. Default is Material.DEFAULT
settings PolyhedronConversionSettings
Conversion settings. Default is an empty object
Mesh.merge(meshes)Mesh
Creates a new mesh containing all the pieces of a collection of meshes
Parameters
meshes Mesh[]
The meshes to combine
interface PolyhedronConversionSettings
This is an interface representing how a Polyhedron should be converted into a mesh. This is not a class, but rather describes the structure of an object that must be used in certain cases.
Properties
smooth? Boolean
Whether the normals should be smoothed across vertices. Default is false
smoothLimit? Number
The minimum dot product between normals of adjacent faces required for them to be smoothed together. This has no effect is .smooth is false. Default is 0 (faces beyond perpendicular aren't smoothed)
