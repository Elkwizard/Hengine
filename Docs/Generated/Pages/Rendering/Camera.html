interface BaseCamera
Represents a camera in a scene. This class should not be constructed and is available via the .camera property of Scene. The transformation represented by this matrix is from screen-space to world-space.
Implementers
Camera2D, Camera3D
Properties
zoomNumber
The magnification level of the camera
rotationNumber
The clockwise roll (in radians) of the camera. Starts at 0
Methods
restoreZoom()void
Sets the zoom to 1.
zoomIn(amount)void
Zooms in by a specified amount.
Parameters
amount Number
The amount to zoom in by
zoomOut(amount)void
Zooms out by a specified amount.
Parameters
amount Number
The amount to zoom out by
zoomBy(factor)void
Multiplies the current zoom value.
Parameters
factor Number
The amount to multiply the current zoom
screenSpaceToWorldSpace(point)Vector2
Maps a given point from screen-space to world-space.
Parameters
point Vector2
The point to transform
worldSpaceToScreenSpace(point)Vector2
Maps a given point from world-space to screen-space.
Parameters
point Vector2
The point to transform
class Camera2D extends Matrix3 implements BaseCamera
Represents the camera in a 2D scene.
Properties
positionVector2
The current center of the camera's view. This starts as new Vector2(width / 2, height / 2)
Methods
get screenRect
Returns the axis-aligned bounding box of the current viewport.
rotateTowards(angle, ferocity)void
Smoothly moves the camera toward a new rotation value.
Parameters
angle Number
The new rotation to move toward (in radians)
ferocity Number
The degree to which the camera should move toward the new position, on [0, 1]
moveTowards(point, ferocity)void
Smoothly moves the camera toward a new position value.
Parameters
point Vector2
The new position to move toward
ferocity Number
The degree to which the camera should move toward the new position, on [0, 1]
constrain(boundary | x, y, width, height)void
Moves the camera such that the entire viewport is entire a given axis-aligned rectangular boundary. If the boundary is smaller than the viewport, the behavior is undefined.
Parameters
boundary Rect
The boundary in which the camera's viewport must exist
Parameters
x Number
The x coordinate of the upper-left corner of the boundary
y Number
The y coordinate of the upper-left corner of the boundary
width Number
The width of the boundary
height Number
The height of the boundary
zoomAbout(center, factor)void
Zooms in/out about a specific point (in world-space) by a specific factor.
Parameters
center Vector2
The zoom center
factor Number
The zoom multiplier
drawInWorldSpace(render)void
Assuming the renderer is currently in screen-space, transforms to world-space, calls a rendering function, and then transforms back to screen-space.
Parameters
render () => void
The function to call while in the world-space context
drawInScreenSpace(render)void
Assuming the renderer is currently in world-space, transforms to screen-space, calls a rendering function, and then transforms back to world-space.
Parameters
render () => void
The function to call while in the screen-space context
class Camera3D extends Matrix4 implements BaseCamera
Represents the camera in a 3D scene.
Properties
positionVector3
The location of the camera, in world-space
directionVector3
The direction the camera is facing. This must be a unit vector
