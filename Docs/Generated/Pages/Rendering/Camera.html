class Camera extends Matrix3
Represents the camera in a scene. This class should be constructed and is available via the .camera property of Scene. The transformation represented by this matrix is from screen-space to world-space.
Properties
positionVector2
The current center of the camera's view. This starts as new Vector2(width / 2, height / 2)
rotationNumber
The clockwise roll (in radians) of the camera. Starts at 0
zoomNumber
The zoom factor of the camera. Starts at 1
Methods
get screenRect
Returns the axis-aligned bounding box of the current viewport.
rotateTowards(angle, ferocity)void
Smoothly moves the camera toward a new rotation value.
Parameters
angle Number
The new rotation to move toward (in radians)
ferocity Number
The degree to which the camera should move toward the new position, on [0, 1]
moveTowards(point, ferocity)void
Smoothly moves the camera toward a new position value.
Parameters
point Vector2
The new position to move toward
ferocity Number
The degree to which the camera should move toward the new position, on [0, 1]
constrain(boundary | x, y, width, height)void
Moves the camera such that the entire viewport is entire a given axis-aligned rectangular boundary. If the boundary is smaller than the viewport, the behavior is undefined.
Parameters
boundary Rect
The boundary in which the camera's viewport must exist
Parameters
x Number
The x coordinate of the upper-left corner of the boundary
y Number
The y coordinate of the upper-left corner of the boundary
width Number
The width of the boundary
height Number
The height of the boundary
