class MeshRenderer
Provides a set of methods for rendering meshes in various ways in a scene. This class should not be constructed, and should be accessed via Artist3D.prototype.mesh.
Methods
default(castShadows?)void
Renders the current mesh at the current transform origin.
Parameters
castShadows Boolean
Whether the mesh should cast shadows. If the same mesh is rendered multiple times, any of them choosing to cast shadows will lead to all of them casting shadows. Default is true
at(transform, castShadows?)void
Renders the current mesh at a given transform relative to the current transform.
Parameters
transform Matrix4
The transform at which the mesh will be rendered
castShadows Boolean
Whether the mesh should cast shadows. If the same mesh is rendered multiple times, any of them choosing to cast shadows will lead to all of them casting shadows. Default is true
instance(transforms, castShadows?)void
Renders the current mesh at a variety of different transforms relative to the current transform.
Parameters
transforms Matrix4[]
The transformations to use for each instance
castShadows Boolean
Whether the mesh should cast shadows. If the same mesh is rendered multiple times, any of them choosing to cast shadows will lead to all of them casting shadows. Default is true
class LightRenderer
Provides a set of methods for creating different types of lights in a 3D scene. This class should not be constructed, and should be accessed via Artist3D.prototype.light.
Methods
ambient()void
Queues an ambient light to be rendered. This light will affect all objects from all sides equally.
point(position)void
Queues an ambient light to be rendered. This light will affect all objects outwards from a specific source.
Parameters
position Vector3
The location of the point light
directional(direction, shadow?)void
Queues a directional light to be rendered. This light emits parallel rays from an infinite distance in a given direction.
Parameters
direction Vector3
The direction of the light
shadow Boolean
Whether the light casts shadows. Default is true
class StrokeRenderer3D
Provides a set of methods for rendering lines and outlines in various ways in three dimensions. This class should not be constructed, and should be accessed via Artist3D.prototype.stroke.
Methods
line/arrow(a, b | line)void
Draws a line segment, with an arrow head at the end for the arrow variant.
Parameters
a Vector3
The start of the line segment to draw
b Vector3
The end of the line segment to draw
Parameters
line Line3D
The line segment to draw
shape(polyhedron)void
Draws a 3D wireframe of a given polyhedron
Parameters
polyhedron Polyhedron
The polyhedron to draw a 3D wireframe of
class DrawRenderer3D
Provides a set of methods for rendering solid shapes in three dimensions. This class should not be constructed, and should be accessed via Artist3D.prototype.draw.
Methods
sphere(center, radius | sphere)void
Renders a sphere in a solid color.
Parameters
center Vector3
The center of the sphere
radius Number
The radius of the sphere
Parameters
sphere Sphere
The sphere to render
prism(min, max | prism)void
Renders a rectangular Prism in a solid color.
Parameters
min Vector3
The back-top-left corner of the rectangular prism
max Vector3
The front-bottom-right corner of the rectangular prism
Parameters
prism Prism
The rectangular prism to render
shape(poly)void
Renders an arbitrary Polyhedron in a solid color.
Parameters
poly Polyhedron
The Polyhedron to render
