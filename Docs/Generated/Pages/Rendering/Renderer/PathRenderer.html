class PathRenderer
Represents a generic drawing API of an Artist. The exact operation used to render the paths is specified in subclasses, but this class which shapes are possible and how they are specified. This is an abstract superclass and should not be constructed.
Subclasses
DrawRenderer, StrokeRenderer, ImageRenderer, ClipRenderer
