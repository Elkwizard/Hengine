class ClipRenderer extends PathRenderer
Represents the clipping API of an Artist. This adds various shapes to the current clipping mask. Each path created will be added to the current clipping state, which means that future draw calls will be able to modify the pixels outside the current clipped area.
class ImageRenderer extends PathRenderer
Represents the image drawing API of an Artist. This draws images in various paths. For non-rectangular shapes, the image is scaled to be the size of the shape's bounding box, and then only the portion of the image inside the shape is shown.
