class ShaderMaterial extends Material
Represents a custom shader-based material with a user-specified vertex and fragment shader programs.
Properties
uniforms Object
A mapping from uniform names to uniform values, with the type correspondence specified in the GLSL API. This can be used to customize the behavior of the material
Methods
constructor(fragmentShader?, vertexShader?)
Creates a new custom material.
Parameters
fragmentShader String
The source code for the fragment shader. The default produces an opaque white color under all circumstances
vertexShader String
The source code for the vertex shader. The default projects the vertex position and passes no additional information to the fragment shader
