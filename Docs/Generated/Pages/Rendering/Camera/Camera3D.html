class Camera3D extends Matrix4 implements Camera
Represents the camera in a 3D scene.
Properties
positionVector3
The location of the camera, in world-space. Starts at (0, 0, 0)
directionVector3
The direction the camera is facing. This must be a unit vector, and starts as (0, 0, 1)
pcMatrixMatrix4
The product of the camera's projection matrix and itself. This property is read-only and only updates when cacheScreen() is called
Methods
constructor(canvas)
Creates a new camera at (0, 0, 0) pointing toward the positive z axis.
Parameters
canvas ImageType
The surface to target
get screenFrustum
Returns the world-space frustum of the camera, and synchronizes .screen and .pcMatrix to match the location and orientation of the camera.
get projectionMatrix4
Returns the projection matrix of the camera. This may vary as the dimensions of the rendering surface change.
set projection(projection)
Configures the camera to use a custom projection.
Parameters
projection Matrix4
The homogenous projection matrix to use
perspective(fov, zNear, zFar)void
Configures the camera to use a perspective projection.
Parameters
fov Number
The field of view of the camera in radians
zNear Number
The location of the near clipping plane
zFar Number
The location of the far clipping plane
orthographic(span, depth)void
Configures the camera to use a orthographic projection.
Parameters
span Number
The size of the x and y spans included in the projection
depth Number
The maximum depth included in the projection
look(xAngle, yAngle)void
Points the camera in a specific direction, with a specified angle from +z on the horizontal and vertical axes. The vertical input is clamped to avoid gimbal lock.
Parameters
xAngle Number
The angle in radians about the y axis from +z to the camera direction
yAngle Number
The angle in radians about the x axis from +z to the camera direction
