interface Camera extends Matrix
Represents a camera in a scene targeting a specific rendering surface. The transformation represented by this matrix is from world-space to screen-space.
Implementers
Camera2D, Camera3D
Properties
zoomNumber
The magnification level of the camera
rotationNumber
The clockwise roll (in radians) of the camera. Starts at 0
Methods
restoreZoom()void
Sets the zoom to 1.
zoomIn(amount)void
Zooms in by a specified amount.
Parameters
amount Number
The amount to zoom in by
zoomOut(amount)void
Zooms out by a specified amount.
Parameters
amount Number
The amount to zoom out by
zoomBy(factor)void
Multiplies the current zoom value.
Parameters
factor Number
The amount to multiply the current zoom
screenToWorld(point)Vector2
Maps a given point from screen-space to world-space.
Parameters
point Vector2
The point to transform
worldToScreen(point)Vector2
Maps a given point from world-space to screen-space.
Parameters
point Vector2
The point to transform
