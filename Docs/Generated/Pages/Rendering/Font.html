class Font
Represents a font, including family, side, and styling. Fonts can be used in the text rendering functions of Artist.
Properties
sizeNumber
The size of the font in CSS pixels
familyString
The string identifier for the font family
boldBoolean
Whether the font is bold
italicBoolean
Whether the font is italic
lineHeightNumber
The height of a line of text in the font. This determines spacing between multiline strings
tabSizeNumber
The number of spaces a tab is equivalent to for this font
Font.[FAMILY][SIZE]Font
These are premade fonts of four common families ("Serif", "Arial", "Cursive", "Monospace") of every size that is a multiple of 5 and less than 100. e.g. Font.Arial10, or Font.Monospace95
Methods
constructor(size, family, bold?, italic?)void
Creates a new font. This operation is fairly expensive and thus should not be done every frame unless necessary.
Parameters
sizeNumber
The size of the font
familyString
The font family
boldBoolean
Whether the font is bold, default is false
italicBoolean
Whether the font is italic, default is false
packText(text, maxWidth)String
Packs a string of text into a fixed width, adding new lines as necessary to prevent overflow.
Parameters
textString
The text to pack
maxWidthNumber
The maximum width of a single line in the output text
getTextBounds(text, maxWidth?)void
Returns the width and height of a string of text, optionally after being packed into a fixed max width. The return value contains width and height properties, both of which are Numbers.
Parameters
textString
The text to be measured
maxWidthNumber
The maximum width of a single line, default is Infinity
getTextLineWidth(textLine)Number
Returns the width of a single line of text. This method is faster than .getTextWidth().
Parameters
textLineString
A single line string of text to measure
getTextLineHeight(textLine)Number
Returns the height of a single line of text. This method is faster than .getTextHeight().
Parameters
textLineString
A single line string of text to measure
getTextWidth(text)Number
Returns the width of a string of text.
Parameters
textString
The text to measure
getTextHeight(text)Number
Returns the height of a string of text.
Parameters
textString
The text to measure
toString()String
Converts the Font to a valid CSS font string.
get(void | destination)Font
Copies the font into an optional destination font. If no destination is provided, a new Font is created to hold the copy.
Parameters
destinationFont
The destination to be copied into
