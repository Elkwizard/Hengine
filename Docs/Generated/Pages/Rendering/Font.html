class Font
Represents a font, including family, side, and styling. Fonts can be used in the text rendering functions of Artist. const font = new Font(15, "Consolas", false, true); // italic 15px Consolas renderer.draw(new Color("black")).text(font, "Hello World!", 0, 0);
Properties
sizeNumber
The size of the font in CSS pixels
familyString
The string identifier for the font family
boldBoolean
Whether the font is bold
italicBoolean
Whether the font is italic
lineHeightNumber
The height of a line of text in the font. This determines spacing between multiline strings
tabSizeNumber
The number of spaces a tab is equivalent to for this font
Font.[FAMILY][SIZE]Font
These are premade fonts of four common families ("Serif", "Arial", "Cursive", "Monospace") of every size that is a multiple of 5 and less than 100. e.g. Font.Arial10, or Font.Monospace95
Methods
constructor(size, family, bold?, italic?)
Creates a new font. This operation is fairly expensive and thus should not be done every frame unless necessary.
Parameters
size Number
The size of the font
family String
The font family
bold Boolean
Whether the font is bold, default is false
italic Boolean
Whether the font is italic, default is false
packText(text, maxWidth)String
Packs a string of text into a fixed width, adding new lines as necessary to prevent overflow.
Parameters
text String
The text to pack
maxWidth Number
The maximum allowed width of a single line in the output text
getTextBounds(text, maxWidth?){ width: Number, height: Number }
Returns the width and height of a string of text, optionally after being packed into a fixed max width. The return value contains .width and .height properties, both of which are Numbers.
Parameters
text String
The text to be measured
maxWidth Number
The maximum allowed width of a single line, default is Infinity
getTextLineWidth(textLine)Number

Parameters

getTextLineHeight(textLine)Number

Parameters

Parameters

Parameters
