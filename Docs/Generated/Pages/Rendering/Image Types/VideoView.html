class VideoView extends ImageType
Represents a video that can have the current frame rendered. These should be loaded using HengineVideoResource and not constructed directly.
Properties
loopsBoolean
Whether or not the video loops.
playingBoolean
Whether or not the video is currently playing. This value is read-only
Methods
getFrame()Frame
Returns a copy of the current frame of the video.
pause()void
Pauses playback of the video.
play()void
Resumes playback of the video.
