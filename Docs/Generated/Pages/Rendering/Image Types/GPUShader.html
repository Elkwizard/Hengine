class GPUShader extends ImageType implements GPUInterface
Represents the renderable result of a GLSL fragment shader invoked for every pixel in a rectangular region. The entry point for the shader is a function of the form: vec4 shader() { ... } Several engine-provided uniforms are given, specifically: uniform vec2 position; // the pixel-space coordinates of the pixel, with (0, 0) in the top-left corner uniform vec2 resolution; // the width and height of the image, in pixels // grayscale shader const shader = new GPUShader(300, 300, ` uniform sampler2D image; vec4 shader() { vec2 uv = position / resolution; vec3 color = texture(image, uv); float brightness = (color.r + color.g + color.b) / 3.0; return vec4(vec3(brightness), 1.0); } `); const cat = loadResource("cat.png"); shader.setUniform("image", cat); // put image in shader renderer.image(shader).default(0, 0); // draw grayscale cat
Methods
constructor(width, height, glsl, pixelRatio?)
Creates a new GPUShader.
Parameters
width Number
The width of the rectangle on which the shader is invoked
height Number
The height of the rectangle on which the shader is invoked
glsl String
The GLSL source code of the shader
pixelRatio Number
The pixel ratio of the shader. Higher values will result in more shader invocations. Default is 1
