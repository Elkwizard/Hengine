class GPUShader extends ImageType
Represents the renderable result of a GLSL fragment shader invoked for every pixel in a rectangular region. The entry point for the shader is a function of the form: vec4 shader() { // your main code here } Several engine-provided uniforms are given, specifically: uniform vec2 position; // the pixel-space coordinates of the pixel, with (0, 0) in the top-left corner uniform vec2 resolution; // the width and height of the image, in pixels // grayscale shader const shader = new GPUShader(300, 300, ` uniform sampler2D image; vec4 shader() { vec2 uv = position / resolution; vec3 color = texture(image, uv); float brightness = (color.r + color.g + color.b) / 3.0; return vec4(vec3(brightness), 1.0); } `); const cat = loadResource("cat.png"); shader.setArgument("image", cat); // put image in shader renderer.image(shader).default(0, 0); // draw grayscale cat

class GLSLError extends Error
Represents an error in a GLSL program. These are constructed by GPUShader and GPUComputation but should not be directly constructed.
Properties
lineNumber
The line on which the error occured
descString
The description of the error
Methods
