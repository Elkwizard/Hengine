class ImageType
Represents an image that can be rendered. This is an abstract superclass and should not be constructed.
Subclasses
CanvasImage, Animation, AnimationStateMachine, HImage, Frame, GPUShader, StaticImage, Texture, VideoView, WebcamCapture
Properties
widthNumber
The natural rendered width of the image
heightNumber
The natural rendered height of the image
Methods
get pixelRatioNumber
Returns the pixel density of the image, measured as the ratio of the number of pixels in a row of the image to the natural width of the image.
get pixelWidthNumber
Returns the width of the image in actual pixels, accounting for the pixel density of the image.
get pixelHeightNumber
Returns the height of the image in actual pixels, accounting for the pixel density of the image.
resize(width, height)void
Simultaneously updates the width and height of the image. Updates that occur on image resizing will only happen once with a call to this method, which can improve performance as opposed to simply assigning to .width and .height in a row.
Parameters
width Number
The new natural width of the image
height Number
The new natural height of the image
contains(point | x, y)Boolean
Checks whether a given point is inside the natural bounds of the image with the upper-left corner at the origin. This operation is inclusive on the lower bound and exclusive on the upper bound.
Parameters
point Vector2
The point to check
Parameters
x Number
The x coordinate of the point to check
y Number
The y coordinate of the point to check
inferWidth(height)Number
Returns the width of the image required for a given height to maintain the aspect ratio.
Parameters
height Number
The potential height of the image
inferHeight(width)Number
Returns the height of the image required for a given width to maintain the aspect ratio.
Parameters
width Number
The potential width of the image
toDataURL()String
Converts the content of the image to a data: url.
download(name)Promise<void>
Downloads the image as a PNG, with a specified name. Returns a promise that resolves when the image downloads.
Parameters
name String
The name of the downloaded image, without the extension
