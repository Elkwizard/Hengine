class WebcamCapture extends ImageType
Represents a video feed from a webcam that can be used as an image. These should be loaded using HengineWebcamResource and not constructed directly.
Properties
recordingBoolean
Whether or not the webcam is capturing new frames
Methods
getFrame()Frame
Returns a copy of the current view from the webcam.
pause()void
Stops the webcam from recording.
play()void
Resumes recording from the webcam.
