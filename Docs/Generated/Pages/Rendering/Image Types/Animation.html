class Animation extends ImageType implements Copyable
Represents a frame-by-frame animation. These should be loaded using HengineAnimationResource and should not be constructed directly. For the purposes of the animation, a frame elapses each time the animation is drawn. const catKnead = loadResource("catKnead"); // load the Animation intervals.continuous(() => { renderer.image(catKnead).default(0, 0); // the animation will advance });
Properties
framesImageType[]
The images that make up the animation
autoAdvanceBoolean
Whether or not the animation will advance to the next frame when drawn. Starts as true
loopsBoolean
Whether or not the animation will reset to the beginning after completing
delayNumber
The number of frames each animation frame will be visible for
timerNumber
The current progress (in frames) of the animation
totalTimeNumber
The total amount of frames it will take the animation to complete
onEnd() => void
A function to be called when the animation completes (this will be called even if the animation loops)
Methods
constructor(frames, delay, loops, onEnd?)
Creates a new animation from a given set of frames.
Parameters
frames ImageType[]
The frames of the animation
delay Number
The delay (in frames) between each frame
loops boolean
Whether or not the animation loops
onEnd () => void
A function to be called when the animation completes. Default is a no-op
get doneBoolean
Returns whether or not the animation has completed. If the animation loops, this will always return false.
advance()void
Advances the animation by one update cycle.
stop()void
Pauses playback of the animation.
start()void
Resumes playback of the animation.
reset()void
Restarts the animation.
Animation.fromImage(spritesheet, imgWidth, imgHeight, delay, loops?, onEnd?)Animation
Creates a new animation based on a horizontal sprite sheet containing the frames directly next to each other.
Parameters
spritesheet ImageType
The image containing all the frames
imgWidth Number
The width (in pixels) of each frame
imgHeight Number
The height (in pixels) of each frame
delay Number
The amount of frames each animation frame is visible for
loops Boolean
Whether or not the animation loops. Default is true
onEnd () => void
A function to execute when the animation completes. Default is a no-op
