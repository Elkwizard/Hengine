class AnimationStateMachine extends ImageType
Represents a collection of animations that play at different times. This is modelled using a collection of states (which can be of any type(s)), that can have different associated animations. This also allows for transition animations between states.
Methods
constructor(stateAnimations, initialState)
Creates a new AnimationStateMachine.
Parameters
stateAnimations Map
The Animations corresponding to each state.
initialState Any
The initial state
get stateAny
Returns the current state of the state machine.
set state(state)
Sets the state of the state machine. If the specified state's animation doesn't loop, then the state will revert back after the animation completes.
Parameters
state Any
The new state of the state machine
addTransition(initial, final, animation)void
Adds an animation to be played when transitioning between two specified states.
Parameters
initial Any
The state being transitioned from
final Any
The state being transitioned to
animation Animation
The animation to play during this time. .loops must be false
