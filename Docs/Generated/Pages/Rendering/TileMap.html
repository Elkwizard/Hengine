class TileMap
Represents a collection of sprites contained within a single image. const spriteSheet = loadResource("sprites.png"); // 32 x 32 const tileMap = TileMap.regular(spriteSheet, 16, 16, ["cat", "dog", "fish", "creature"]); // extract 4 tiles renderer.image(tileMap.get("fish")).rect(0, 0, 64, 64);
Methods
constructor(image, sections)
Creates a new TileMap.
Parameters
image ImageType
The overall sprite sheet
sections { name: Any, area: Rect }[]
The regions of the sprite sheet assigned to each sprite. Each entry in this list has a .name property, which is any unique identifier for the sprite, and a .area property representing the natural region of the sprite sheet assigned to this name
get(name)ImageType
Retrieves a specific sprite from the sprite sheet.
Parameters
name Any
The identifier of a specific sprite
TileMap.regular(image, tileWidth, tileHeight, names)TileMap
Creates a tile map based on a series of contiguous, fixed-size, left-to-right, top-to-bottom tiles.
Parameters
image ImageType
The sprite sheet
tileWidth Number
The width of each tile
tileHeight Number
The height of each tile
names Any[]
The list of identifiers for each tile
