GLSL API
The version of GLSL used in the Hengine is GLSL ES 3.0. Interactions between GLSL and javascript (such as GPUShader and GPUComputation) require associations between types in both the languages. These associations are laid out in the following table:
GLSL TypeJS Type
int, uint, floatNumber
boolBoolean
vec2, ivec2, uvec2Vector2
vec3, ivec3, uvec3Vector3
ivec4, uvec4Vector4
vec4Vector4, Color
sampler2DImageType
structObject
arrayArray
