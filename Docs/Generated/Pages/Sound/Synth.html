class Synth
Represents a synthesizer which can create pure tones. const synth = new Synth(); synth.playSequence([ { frequency: 440, duration: 1000 }, // one second A note { duration: 500, volume: 0 } // half second pause { note: "A", duration: 1000 } // one second A note ]);
Methods
constructor()
Creates a new Synth.
play(tone)((Number) => void)/Promise<void>
Plays a tone. If the tone has a specified duration, this returns a promise that resolves when it completes. Otherwise, this returns the .stop() method of the SynthChannel playing this tone.
Parameters
tone Tone
The specification of the tone to play.
async playSequence(all, globals?)void
Plays a sequence of tones with specified durations. Returns a promise that resolves when they have all completed.
Parameters
all Tone[]
The specifications of the tones to play. See the class description
globals Tone
All the properties of this object will be used as defaults for the properties of tones specified in all. Default is { }
class Tone
This is not an actual class, but rather an interface for tone specifications.
Properties
duration?Number
The duration (in milliseconds) of the tone. If not specified, the tone can be stopped at will
frequency?Number
The frequency of the tone in Hertz. If not specified, .note must be
note?String
The letter name of the the note, optionally with a "#" appended for sharp or a "b" for flat. If not specified, .frequency must be
octave?Number
Which octave on a piano the note would appear in. Default is 4. This is ignored if .frequency is specified
volume?Number
The volume of the tone on [0, 1]. Default is 1
wave?String
The name of the waveform for the tone. This is one of ("sine", "sawtooth", "triangle", "square"). Default is "sine"
class SynthChannel
Represents a channel on which tones can be played.
Methods
stop(wait?)Promise<void>
Stops the tone from playing, optionally after a delay. Returns a promise that resolves when the tone stops.
Parameters
wait Number
The interval (in milliseconds) to wait before stopping the sound. Default is 0
