class Sound
Represents an external sound clip. These should be loaded using HengineSoundResource and not constructed directly.
Properties
loopsBoolean
Whether or not the sound loops upon completion. This value is read-only
durationNumber
The length, in milliseconds, of the sound
Methods
play(volume)SoundInstance
Starts playback of the sound in a new instance, which is returned. Multiple sound instances can play at once.
Parameters
volume Number
The volume of the playback on [0, 1]
class SoundInstance
Represents an ongoing playback of a Sound.
Properties
volumeNumber
The volume of the playback on [0, 1]
Methods
get timeNumber
Returns the current time in milliseconds since playback began.
stop()void
Stops playback of the sound.
