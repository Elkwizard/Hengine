class MathObject implements Copyable
Represents a mathematical object, on which operations (+, -, *, /, %, **) can be performed. The operations are provided as two sets of methods, one which mutates the caller, and one which creates a new object to hold the result. new Complex(3, 6).over(3) // 1 + 2i new Vector3(1, 2, 3).plus(3) // (4, 5, 6) Quaternion.fromRotation(Vector3.up, Math.PI) // 0 + 0i + -1j + 0k This is an abstract superclass and should not be constructed.
