class SceneObject extends SceneElement
Represents an object in a Scene. These can be created by the .add...Element() methods of ElementContainer. Every scene object has a collection of local-space shapes that make up its presence. These shapes are used for culling, rendering, and physics hitboxes.
Subclasses
