class Operable
Represents a composite mathematical object which element-wise operations can be performed on. Many mathematical operations (plus, minus, times, over) can be performed on operables of any subclass type, allowing for convenient polymorphism. All immutable methods, including static methods, are available on Number. This means that Number can largely be considered a subclass of operable and can be used for operable-typed arguments. This is an abstract superclass and should not be constructed.
