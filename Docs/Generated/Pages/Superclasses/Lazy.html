class Lazy
Facilitates the creation of lazily-evaluated expressions.
Methods
Lazy.define(obj, name, calculateValue)void
Defines a lazy-evaluated property of an object. The first access to the property will calculate the result, which will be cached and returned on all future accesses.
Parameters
obj Object
The target object for the property
name String
The key for the property
calculateValue Function
The no-argument function used to initially compute the value.
