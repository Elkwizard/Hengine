class Lazy
Facilitates the creation of lazily-evaluated expressions. const object = { }; Lazy.define(object, "lazy", () => { // an expensive operation let result = 0; for (let i = 0; i < 1e6; i++) result += Random.random(); return result; }); console.log(object.lazy); // takes a while console.log(object.lazy); // cheap from now on
Methods
Lazy.define(obj, name, calculateValue)void
Defines a lazy-evaluated property of an object. The first access to the property will calculate the result, which will be cached and returned on all future accesses.
Parameters
obj Object
The target object for the property
name String
The key for the property
calculateValue () => Any
The no-argument function used to initially compute the value.
