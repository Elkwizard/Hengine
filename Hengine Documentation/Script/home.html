Script

The Script class is used for storing a collection of custom methods that can be applied to objects to make them act like the Script.

Properties:

  • methods

    An Object containing all of the methods in the Script.


Methods:

  • constructor(String name[, Object fns])

    Initializes the Script with the name name. If fns is provided, Script.addMethod will be called for each methods of the object with the name of the method as the first and last arguments and the function as the middle argument.

  • run()

    Calls every function in Script.methods. Not especially useful.

  • addTo(Object obj, ANY ...args)

    Adds each method in Script.methods to an object stored in obj[Script.name]. The methods will be bound to obj. Any methods with the flag 'init' will be called with the arguments args.

  • addMethod(String name, Function method, String flag)

    Adds a method to Script.methods with the name of name, with a flag of flag.

  • removeMethod(String name)

    Deletes method name from Script.methods.
