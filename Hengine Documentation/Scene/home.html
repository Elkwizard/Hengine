Scene

The Scene class is used to manage and modify SceneObjects. Scene inherits from InactiveScene.


Properties:

  • c

    Scene.c is an Artist object. This is the same Artist instance used in the Project.

  • home

    Scene.home is an Engine object. This is the same Engine instance used in the Project. This is also the engine that created the Scene.

  • display

    Scene.display is a Rect object containing the x, y, width, and height of the part of the Scene that is displayed on the canvas.

  • cullGraphics

    A boolean indicating whether or not PhysicsObjects should check if they are onscreen before being drawn. Default is true.

  • zoom

    A float indicating the level of zoom used when displaying the Scene.
    Example:g.console.log(s.zoom); // 1 s.zoomOut(0.1); g.console.log(s.zoom); // 0.9090909090


Methods:

  • constructor(String name, Artist context, float gravity, float airResistance, Engine home)

    Calls InactiveScene.prototype.constructor(gravity, gravity, airResistance). Scene.home is initialized to home. Scene.c is initialized to context.

  • drawWithTransformations(Function draw)

    Adjusts and translates the canvas to fit the current display location and then calls draw.

  • adjustPointForDisplay(Vertex p)

    Adjusts the coordinates of p from a point on the screen to a point in the Scene.

  • updateDisplayAt(float x, float y, float width, float height)

    Moves the display to show the part of the Scene at (x, y) at the upper left corner. The display has a width of width and a height of height.

  • restoreZoom()

    Resets the zoom level to 1.

  • zoomIn(float amount)

    Zooms the display in on the Scene by a factor of amount.

  • zoomOut(float amount)

    Zooms the display away from the Scene by a factor of amount.

  • restoreZoom()

    Resets the zoom level to 1.

  • loadScene(InactiveScene sc)

    Loads all of the elements from sc into the Scene.
