Controls

Properties:

  • up

    The key associated with 'up'.

  • down

    The key associated with 'down'.

  • left

    The key associated with 'left'.

  • right

    The key associated with 'right'.

  • interact1

    The first generic interaction key.

  • interact2

    The second generic interaction key.


Methods:

  • constructor([up[, down[, left[, right[, interact1, [interact2]]]]]])

    Each property is assigned to the parameter with the same name. eg: Controls.down = down.
    Example:let controls = new Controls("w", "s", "a", "d") // creates WASD controls
