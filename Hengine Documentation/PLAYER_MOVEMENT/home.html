PLAYER_MOVEMENT

The PLAYER_MOVEMENT constant is used as an alternative to Scene.prototype.giveElementSubjectiveMovement. Although it has the same functionality, it allows for more flexibility in terms of customizing the Object it is attached to.
Implementation: 

const PLAYER_MOVEMENT = new ElementScript("movement");
PLAYER_MOVEMENT.addMethod("init", function(){
	if(!this.controls.up){
		this.controls = new Controls("w", "s", "a", "d");
	}
	this.applyGravity = true;
}, "init");
PLAYER_MOVEMENT.addMethod("update", function(){
	if (K.P(this.controls.down)) this.speed.y += this.accel.y;
	if (K.P(this.controls.left)) this.accel.x = -0.1;
	else if (K.P(this.controls.right)) this.accel.x = 0.1;
	else this.accel.x = 0;
	if (K.P(this.controls.up)) {
		if (this.colliding.bottom) {
			this.speed.y = -5;
		}
	}
}, "update");


Methods:

PLAYER_MOVEMENT is an ElementScript object. It can be applied to a PhysicsObject by calling PLAYER_MOVEMENT.addTo.
