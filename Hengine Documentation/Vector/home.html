Vector

The Vector class is used for storing positions in space. The Vector classes are only available if the 'Vector.js' script is enabled. There is no actual Vector class, but there are Vector1, Vector2, Vector3, and Vector4 for storing positions in 1, 2, 3, and 4 dimensions. A Vertex is the same as a Vector2.
Example:let pos1 = new Vertex(5, 10); let pos2 = new Vector2(5, 50); g.console.log(pos1.x) // 5 g.console.log(pos2.x) // 5 Note: all methods that a Vertex as an argument can also take any Object with 'x' and 'y' properties. These could be things such as SceneObjects, M, or a Vector4.

Properties:

  • x

    Stores a value. Available in all Vectors.

  • y

    Stores a value. Available in Vectors 2 and up.

  • z

    Stores a value. Available in Vectors 3 and up.

  • w

    Stores a value. Available only for Vector4.


Methods:

  • constructor(x[, y[, z[, w]]])

    Sets the corresponding property to each parameter.
    Example:let pos = new Vector2(25, 20); // creates new Vector2 g.console.log(pos.x); // 25

  • add(float or Vector v)

    If v is a number, it adds v to each property of the Vector. If v is a Vector, then it adds the corresponding element to each element of the Vector. Returns the Vector.

  • sub(float or Vector v)

    If v is a number, it subtracts v from each property of the Vector. If v is a Vector, then it subtracts the corresponding element from each property of the Vector. Returns the Vector.

  • mul(float or Vector v)

    If v is a number, it multiplies v by each property of the Vector. If v is a Vector, then it multiplies the corresponding element by each of the Vector. Returns the Vector.

  • div(float or Vector v)

    If v is a number, it divides v by each property of the Vector. If v is a Vector, then it divides each of element the Vector by the corresponding element of v. Returns the Vector.

  • normalize()

    Normalizes the vector.
    Example:

    
let pos = new Vector2(4, 6); // creates new Vector2
g.console.log(pos); // "(4, 6)"
pos.normalize();
g.console.log(pos); // "(0.4, 0.6)"

  • Other Methods

    The Vector2 class specifically has two additional methods:

    • static fromAngle(float angle)

      Returns a new Vector2 created with a direction equal to the angle angle provided in radians.

    • getAngle()

      Returns the direction that the Vector2 is facing as an angle in radians.
