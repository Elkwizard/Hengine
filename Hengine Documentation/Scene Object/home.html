SceneObject

SceneObject inherits from Rect. The SceneObject class is really only meant to be used by the Scene and Inactive Scene classes. SceneObjects are what make up the actual game and would be used for things like health bars and trees. SceneObjects do not collide with other SceneObjects or PhysicsObjects.


Properties:

  • name

    The name of SceneObject.

  • tag

    The tag of SceneObject.

  • home

    A link to the Scene or InactiveScene that created the SceneObject.

  • controls

    The Controls object that contains the keys that influence the SceneObject.

  • hidden

    A boolean indicating whether or not the SceneObject is drawn each frame.

  • log

    An Array of functions representing all the changes that have occured on this SceneObject since it was created.

  • isDead

    A boolean indicating whether or not the SceneObject has been removed from its home scene.

  • response.input

    An Object containing an 'up', 'down', 'left', and 'right' properties, where each value is a function. These functions will be called whenever the corresponding key in the ScenObject.controls Object is pressed.

  • hovered

    A boolean indicating whether or not the mouse is hovering over the SceneObject.

  • custom

    An empty Object for use as you please.

  • scripts

    An Object containing all of the scripts attached to the SceneObject with the keys being the scripts' name.

  • collider

    A RectCollider object containing the hitbox of the SceneObject.

  • lifeSpan

    An integer indicating how many frames the SceneObject has existed for.

  • layer

    A number respresenting the layer the SceneObject is on. A higher layer will be renderered later.

  • hasPhysics

    A boolean indicating whether or not the SceneObject is also a PhysicsObject


Methods:

  • constructor(String name, float x, float y, float width, float height, Controls controls, String tag)

    Calls Rect.prototype.constructor with parameters (x, y, width, height). A new SceneObject is initialized with a name of name. The SceneObject has controls of controls and a tag of tag.

  • response.click(Vertex e)[1]

    e represents the position of the mouse adjusted for the display. This method is called when the mouse clicks on the SceneObject.

  • response.rightClick(Vertex e)[1]

    e represents the position of the mouse adjusted for the display. This method is called when the mouse right clicks on the SceneObject.

  • response.hover(Vertex e)[1]

    e represents the position of the mouse adjusted for the display. This method is called when the mouse enters the SceneObject.

  • draw()[1]

    This method is called every frame.

  • update()[1]

    This method is called every physics update.

  • hide()

    Hides the SceneObject.

  • show()

    undoes SceneObject.prototype.hide.

  • collidePoint(Vertex p)

    Returns a boolean indicating whether or not p is inside of the SceneObject.

  • remove()

    Removes the SceneObject from the Scene.

  • position(Vertex p)

    Sets the position of the SceneObject to p and then calls SceneObject.updateHitbox.

  • mod(Function func)

    Calls func and then adds func to the log.

  • logMod(Function func)

    Adds func to the log.

  • runLog(SceneObject el)

    Binds and runs each method in the log on el.

  • [1]: This method starts empty. It is meant to be set by the user.
