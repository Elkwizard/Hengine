Directions

The Directions class is used to indicate the directions in which things can travel.

Properties:

  • up

    A boolean indicating whether or not things can travel up.

  • down

    A boolean indicating whether or not things can travel down.

  • left

    A boolean indicating whether or not things can travel left.

  • right

    A boolean indicating whether or not things can travel right.

  • prec

    A float between 0 and 1 representing how much things should move in the directions specified, where 1 is not at all and 0 is completely.


Methods:

  • constructor(boolean up, boolean down, boolean left, boolean right, float prec)

    Sets the corresponding properties to the parameters.

  • fixV(float verticalSpeed)

    Returns verticalSpeed adjusted to go in the directions specified by the object.

  • fixH(float horizontalSpeed)

    Returns horizontalSpeed adjusted to go in the directions specified by the object.
