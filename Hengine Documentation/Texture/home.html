Texture

The Texture class is used for storing pixel data and modifying pixels in an optimized way. Textures are often used for shaders.

Properties:

  • pixels

    A 2D array of Colors, representing the pixels in the Texture.


Methods:

  • constructor(int width, int height)

    Creates a new Texture with dimensions of width by height.

  • setPixel(int x, int y, Color color)

    Sets the pixel at (x, y) to the color color.

  • updateImageData()

    Updates the underlying API data, allowing changes made with setPixel to be shown to the screen.

  • [Symbol.iterator]*()

    You can iterate through a Texture, where each item is an array of x and y. You can use object destructuring to make simple shaders with this. Example: 

    const TEX = new Texture();
for (let [x, y] of TEX) {
    let col = Color.random();
    TEX.setPixel(x, y, col);
}
