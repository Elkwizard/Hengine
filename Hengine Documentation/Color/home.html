Color

The Color class is used to parse and store colors. It can be used as a color argument for the Artist's drawing methods.

Properties:

  • red

    The amount of red in the Color, between 0 and 255.

  • green

    The amount of green in the Color, between 0 and 255.

  • blue

    The amount of blue in the Color, between 0 and 255.

  • alpha

    The opacity in the Color, between 0 and 1.


Methods:

  • constructor([int r, int g, int b[, int a]] or [String rgbaRep] or [String hexRep])

    Will use the parameters provided to initialize the color. If a is not provided, then it will count as 1.
    Example:let color = new Color("#40e"); g.console.log(color.get_RGBA()); // "rgba(68, 0, 238, 1)"

  • get_RGBA()

    Returns the rgba representation of the Color as a String.

  • get_HEX()

    Returns the hexidecimal representation of the Color as a String.

  • add(Color or number op)[1]

    Adds op

  • sub(Color or number op)[1]

    Subtracts op

  • mul(Color or number op)[1]

    Multiplies by op

  • div(Color or number op)[1]

    Divides by op

  • mod(Color or number op)[1]

    Performs modulus with the Color and op, where each channel is set to the result.

  • static add(Color or number op, Color or number op2)[1]

    Returns the sum of op and op2.

  • static sub(Color or number op, Color or number op2)[1]

    Returns the difference between op and op2.

  • static mul(Color or number op, Color or number op2)[1]

    Returns the product of op and op2.

  • static div(Color or number op, Color or number op2)[1]

    Returns the quotient of op and op2.

  • static mod(Color or number op, Color or number op2)[1]

    Performs modulus with op and op2, and returns a Color where each channel is set to the result.

  • static grayScale(float per)

    Returns a Color with a grayScale value that is per of the way between black and white.
    Example: g.console.log(Color.grayScale(0).get_RGBA()); // rgba(0, 0, 0, 1) g.console.log(Color.grayScale(1).get_RGBA()); // rgba(255, 255, 255, 1)

  • static lerp(Color op, Color op2, float per)

    Returns a Color with a grayScale value that is per of the way between op and op2.

  • static random()

    Returns a psuedo-random Color.

  • [1]: does the indicated operation with the Color and op for each channel. If op is a number, then op is added to every channel.
