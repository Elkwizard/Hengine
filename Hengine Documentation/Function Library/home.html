FunctionLibrary

The FunctionLibrary class is mainly used by the Engine class to give the user helpful functions. FunctionLibrary is a global class in the Hengine as long as the 'Helper Functions.js' script is enabled.


Methods:

  • constructor()

    Instantiates the FunctionLibrary.

  • countProperties(Object obj)

    Returns the number of properties in obj.

  • objToAry(Object obj)

    Returns an array of the values of properties in obj.

  • getDistance([float x1, float y1, float x2, float y2] or [Vertex p1, Vertex p2])

    Returns the distance between (x1, y1) and (x2, y2) or p1 and p2, if provided.

  • getDistance3D(Vector3 p1, Vector3 p2)

    Returns the distance between p1 and p2 in 3D space.

  • getDistance4D(Vector4 p1, Vector4 p2)

    Returns the distance between p1 and p2 in 4D space.

  • extend(float a, float b)

    Returns a extended away from zero by b.

  • contract(float a, float b)

    Returns a moved towards zero by b.

  • randIntVal(int min, int max)

    Returns a psuedo-random integer between min and max.

  • randFloatVal(int min, int max)

    Returns a psuedo-random float between min and max.

  • randBool()

    Returns a psuedo-random boolean value.

  • randString(int length)

    Returns a psuedo-random String with a length of length.

  • parseBool(String str)

    Returns a boolean parsed from str.

  • parseArray(String str)

    Returns an Array parsed from str.

  • round(float x, int places)

    Returns x, rounded to the nearest placesth.
