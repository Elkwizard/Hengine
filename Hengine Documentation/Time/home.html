Time

Time is a global class in the Hengine as long as the 'Time.js' script is enabled. The time class is used for getting the current time and date, as well as formatting it.


Methods:

  • static getTime()

    Returns an object with seconds, minutes, hours, days, and years properties. Each of these properties contain the associated value as an integer. Note: the years property of the return value is AD. Example:g.console.log(Time.getTime()); // [object Object] g.console.log(Time.getTime().seconds); // 53

  • static getFormattedTime()

    Returns a string containing the current time formatted as "hh:mm:ss" with "AM" or "PM" appended appropriately. Example:g.console.log(Time.getFormattedTime()); // "9:02:53 AM"
