ElementScript

ElementScript inherits from Script. ElementScripts store a collection of custom methods that can be applied to SceneObjects.


Methods:

  • constructor()

    Inherited from Script.

  • addTo(SceneObject el, ANY ...args)

    Attaches the ElementScript to el.scripts and binds it to el. 'init' functions will be called with arguments args.

Tags:

These are things that can be passed as the tag parameter to addMethod

  • "init"

    Is called when ElementScript.prototype.addTo is called. Called with the arguments passed as the arguments after the SceneObject.

  • "update"

    Is called every physics update (every 16ms) and is used for checking and updating values.

  • "draw"

    Is called after SceneObject.prototype.draw and can be used to draw the SceneObject.

  • "collide-"

    This tag followed by a direction ex) "collide-right", "collide-general" will be called when the element collides on that side.
