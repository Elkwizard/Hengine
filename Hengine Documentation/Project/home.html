Project

The Project class is used for setting up the Hengine in a Package. When a Hengine 'Project.html' file is opened, an Project instance is created and stored inside of 'PROJECT'.


Properties:

  • g[1]

    An Engine object. This is the Engine instance referenced throughout the Documentation.

  • s[1]

    A Scene object. This is the same as Project.g.scene.

  • c[1]

    An Artist object. The same as Project.g.renderer.

  • C[1]

    A link to Project.c.c.

  • cl[1]

    A ColorLibrary object. Contains common colors as properties.
    Example:g.console.log(cl.BLACK.get_RGBA()); //"rgba(0, 0, 0, 1)"

  • custom[1]

    An empty object used for storing any custom values associated with the project.


Methods:

  • constructor([String wrapperID, int width, int height, float airResistance, float gravity, String canvasID])

    Creates an Engine with parameters (wrapperID, width, height, airResistance, gravity, canvasID).

  • loadImage(String source)[1]

    returns a HTMLImageElement object with a source of source, where source is a position relative to Package/Art/Sprites.

  • loadSound(String source)[1]

    returns a HTMLAudioElement object with a source of source, where source is a position relative to Package/Sound.

  • rand()[1]

    returns Math.random().

  • middle()[1]

    returns a Vertex object containing the middle of the display/canvas.

  • [1]: This property is also available in the global scope. eg: Project.c = Window.c.
