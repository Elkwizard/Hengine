M

The M constant is used for managing mouse input and movement.

Properties:

  • down

    A boolean indicating whether or not the mouse is currently held down.

  • x

    The current x coordinate of the mouse, relative to the top left corner.

  • y

    The current y coordinate of the mouse, relative to the top left corner.


Listeners:

The M constant also has the properties:

  1. onDown

  2. onUp

  3. onMove

  4. onClick

  5. onRight

  6. onScroll

  7. onDown

Each of these are empty Listeners that will be called. They are called when the corresponding event occurs and passed the Event object as a parameter. These are meant to be set by the user.
