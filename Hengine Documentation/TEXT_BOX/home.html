TEXT_BOX

The textbox script is used for creating textboxes on UI elements for entering text. SceneObjects with TEXT_BOX added to them are very similar to HTMLInputElements in the sense that outwardly they work the same, and, the value of the textbox can be aquired through the property 'value' added to the SceneObject.


Methods:

  • addTo(SceneObject element, Object options)

    The textbox is added on top of the element with the options provided and an empty value. It is not selected.
    Example:

    let textbox = s.addUI("input", 50, 50, 400, 50);
TEXT_BOX.addTo(textbox, {
    fontColor: cl.BLACK,       // sets text color to black
    fontSize: 20,              // sets font size to 20 css pixels
    backgroundColor: cl.WHITE, // sets the background color to white
    borderColor: cl.BLACK      // sets the border color to black
});                            // all of these are optional
