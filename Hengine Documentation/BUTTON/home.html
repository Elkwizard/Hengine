BUTTON

A script that can be used for adding button functionality to SceneObjects by allowing them to easily display text are monitor clicks. What a button does when clicked can be modified by setting the 'onclick' property of the SceneObject to a function to be run. The function should take no parameters

Properties:

  • Property

    Description sample text. Example:g.console.log(expr); // "expr"


Methods:

  • addTo(SceneObject elements, Object options)

    Creates a new textbox with an empty onclick with the options provided.
    Example:

    let button = s.addUI("input", 50, 50, 200, 40);
BUTTON.addTo(button, {
    fontColor: cl.BLUE,       // sets text color to blue
    fontSize: 15,             // sets font size to 15 css pixels
    backgroundColor: cl.RED,  // sets the background color to red
    borderColor: cl.BLUE      // sets the border color to blue
});                           // all of these are optional
