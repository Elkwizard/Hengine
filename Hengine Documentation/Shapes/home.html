Shapes

The Shape classes store values associated with rectangles, triangles, and circles. The draw and stroke methods of the Artist for drawing rectangles and triangles can also take the corresponding Shape object as an argument. There are two shape classes:

  • Rect

    The Rect class stores the position and dimensions of rectangles. Learn more.

  • Triangle

    The Triangle class stores the vertices and position of triangles. Learn more.

  • Circle

    The Circle class stores the position and radius of circles. Learn more.

  • Line

    The Line class stores two end points and midpoint of a line segment. Learn more.
