Animation

The Animation class is used by the Artist class for creating animations.

Properties:

  • frameCount

    The number of frames in the Animation.

  • frames

    An Array of Image objects representing the frames.

  • img

    An Image object representing the current frame.

  • loop

    A boolean indicating whether or not the Animation loops.

  • finResponse

    A function that will be called whenever the Animation loops.

  • delay

    The number of frames between frame advances in the Animation.


Methods:

  • constructor(String src, int frameCount, int delay, boolean loops, Function finResponse)

    Locates the source folder and looks for files named 1.png, 2.png, 3.png and so on. These are used as frames for the Animation which will advance a frame every delay frames. If loops is true, then the Animation will start over when it gets to the end. At the end of each loop, finResponse is called.

  • advance()

    Moves the perceived time of the Animation forward by 1 frame.
