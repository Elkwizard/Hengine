Artist

The Artist class is used for drawing SceneObjects and dealing with images and Animations.

Properties:

  • canvas

    The HTMLCanvasElement object that the Artist is drawing on.

  • c

    The CanvasRenderingContext2D that the Artist uses for drawing.

  • animations

    An array of all Animations created by the Artist.

  • custom

    An empty Object for use as you please.


Methods:

  • constructor(String canvasID, int width, int height)

    Attaches the new Artist to the canvas with ID of canvasID. The canvas' dimensions will be set to height by height, if they are provided. Otherwise, the canvas' dimensions will be preserved.

  • middle()

    Returns a Vertex representing the middle of the canvas.

  • sigmoid(float x)

    Returns the sigmoid of x.

  • simpleCircle(Color fill, Color border)

    Returns a function that when bound to a Rect object draws the Rect as a circle with a background of fill and a border of border.

  • simpleRect(Color fill, Color border)

    Returns a function that when bound to a Rect object draws the Rect as a rectangle with a background of fill and a border of border.

  • createAnimation(String source, int frameCount, int delay, boolean loop, Function finResponse)

    Pushes a new Animation object with parameters (source, frameCount, delay, loop, finResponse) to Artist.animations. Returns the Animation.

  • drawAnimation(Animation anim, float x, float y, float width, float height)

    Draws the current frame of anim at (x, y) with dimensions of width by height. Calls anim.advance.

  • drawImage(Image img, float x, float y, float width, float height)

    Draws img at (x, y) with dimensions of width by height.

  • drawTexture(Texture tex, float x, float y, float width, float height)

    Draws tex at (x, y) with dimensions of width by height. The data in the texture that has been changed since the last updateImageData call will not be shown.

  • draw(Color fill)

    Sets the fill color to fill. Returns an Object with methods:

    • circle([float x, float y, float radius] or [Circle c])

      Fills a circle with a center at (x, y) with a radius of radius, or if provided, the coordinates and radius of c.

    • sector(float x, float y, float radius, float startAngle, float endAngle)

      Fills a sector with a center at (x, y) with a radius of radius, the sector goes from startAngle to endAngle (in radians).

    • rect([float x, float y, float width, float height] or [Rect rect])

      Fills a rectangle with a upper left corner at (x, y) with dimensions of width by height, or, if provided the position and dimensions of rect.

    • triangle([Vertex v1, Vertex v2, Vertex v3] or [Triangle tri])

      Fills a triangle with vertices v1, v2, and v3, or, if provided, with the vertices of tri.

    • text(String font, String text, float x, float y)

      Draws Text text in font font with an upper left corner at (x, y).

  • stroke(Color outline, float lineWidth)

    Sets the outline color to outline. Sets the line width to lineWidth. Returns an Object with methods:

    • circle(float x, float y, float radius)

      Outlines a circle with a center at (x, y) with a radius of radius.

    • sector(float x, float y, float radius, float startAngle, float endAngle)

      Outlines a sector with a center at (x, y) with a radius of radius, the sector goes from startAngle to endAngle (in radians).

    • rect(float x, float y, float width, float height)

      Outlines a rectangle with a upper left corner at (x, y) with dimensions of width by height, or, if provided the position and dimensions of rect.

    • triangle(Vertex v1, Vertex v2, Vertex v3)

      Outlines a triangle with vertices v1, v2, and v3, or, if provided, with the vertices of tri.

    • line([float x1, float y1, float x2, float y2] of [Vertex p1, Vertex p2])

      Draws a line from (x1, y1) to (x2, y2). If provided, it will draw the line from p1 to p2.

  • clear()

    Clears the entirety of the canvas.

  • setBackground(Color background)

    Makes the canvas background background.
