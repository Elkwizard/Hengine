Frame

The Frame class is used for creating Images during runtime. Frame objects can be passed to the Artist classes' drawImage method

Properties:

  • width

    The width of the Frame.

  • height

    The height of the Frame.

  • img

    The Frame represented as an Image.

  • c

    An Artist object that will draw on the Frame.


Methods:

  • constructor(float width, float height)

    Creates a Frame with dimensions width by height.
